# Stage 20A — Frontend Foundation and Visual System

Stage 20A establishes the PoultryPulse web application shell and design
language without changing backend business behavior.

## Visual direction

- Dark-first interface with accessible light mode
- Emerald and teal primary brand colors
- Restrained amber production and finance highlights
- Rounded cards, controls, badges, tables, and floating mobile actions
- Soft depth and subtle background texture
- Responsive sidebar and mobile navigation sheet
- Command search and keyboard access
- Designed loading, error, empty, and placeholder states

## Foundation stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui using Radix primitives
- Geist Sans and Geist Mono
- Lucide icons
- Recharts
- next-themes

## Stage boundary

Stage 20A uses demonstration dashboard data. Stage 20B adds login,
session management, route protection, the authenticated API client, real
farm identity, permission-aware navigation, and live dashboard data.
