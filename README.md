# PoultryPulse

PoultryPulse is a web-based poultry management system designed primarily for layer poultry farms.

## Tagline

Know Your Flock. Grow Your Farm.

## Main Features

- Farm and user management
- Poultry house management
- Layer flock management
- Daily egg production
- Egg inventory
- Feed inventory and consumption
- Mortality and culling
- Vaccination and health management
- Customer management
- Egg sales and payments
- Expense management
- Dashboards, alerts and reports

## Technology Stack

### Backend

- Python
- FastAPI
- SQLAlchemy
- PostgreSQL
- Alembic

### Frontend

- React
- TypeScript
- Tailwind CSS

## Project Structure

- `backend/` — FastAPI application
- `frontend/` — React application
- `database/` — database scripts and seed data
- `docs/` — project documentation
- `scripts/` — development and maintenance scripts
- `deployment/` — production deployment configuration

## Development Status

PoultryPulse is currently under active development.
